Google Drive Sync

Hubungkan Google Drive

Masukkan OAuth 2.0 Client ID dari Google Cloud Console untuk mulai sync otomatis.

Belum punya Client ID? Klik untuk panduan singkat
1
Buka console.cloud.google.com → buat project
2
APIs & Services → Library → aktifkan Google Drive API
3
Credentials → Create Credentials → OAuth 2.0 Client ID → pilih Web application
4
Di Authorized JavaScript origins tambahkan: http://localhost dan null (untuk file://)
5
Copy Client ID → paste di kolom di atas → klik Hubungkan
✅
